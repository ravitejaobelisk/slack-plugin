Leaving this field blank will disable webhook functionality.

This is the webhook endpoint to send requests to, e.g. webhook, slackwebhook.

The webhook endpoint is mapped to the root address of your Jenkins instance and needs to be publicly accessible.

Example:

If your Jenkins instance is http://myjenkins.example.com and your webhook endpoint is slackwebhook, you would set your Slack outgoing webhook url to:

http://myjenkins.example.com/slackwebhook/.

Note that the trailing slash is required.
